# Telegram Leave Bot

## Deployment on Railway

1. Create a new Railway Node.js project.
2. Upload this project folder.
3. Set environment variables:
   - BOT_TOKEN: your Telegram bot token
   - ADMIN_IDS: comma-separated admin Telegram IDs, e.g. 123456789,987654321
4. Railway will automatically install dependencies and run worker.js
5. Use /start in your Telegram group to begin.
